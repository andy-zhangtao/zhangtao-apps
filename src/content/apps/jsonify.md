# JSONify - 专业 JSON 处理工具

专为 macOS 开发者设计的原生 JSON 处理应用，让复杂的数据格式化、语法检查和差异对比变得简单高效。

## 最新版本亮点

### v1.9.0 重大更新
- **性能大幅提升**：大文件处理速度提升 300%
- **界面全新升级**：采用现代化设计语言，视觉体验更佳
- **批量文件支持**：拖拽导入，同时处理多个 JSON 文件
- **智能状态保存**：工作空间自动恢复，无缝衔接工作流程

支持 macOS 11.0+，完美适配 Intel 和 Apple Silicon 芯片。
